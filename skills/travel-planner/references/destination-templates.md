## 行程模板

### 一日游模板

```
上午 (9:00-12:00)
- 主要景点 (2-3小时)

午餐 (12:00-13:30)
- 当地特色餐厅

下午 (13:30-18:00)
- 次要景点 (2小时)
- 休息/咖啡 (1小时)
- 购物/漫步 (1.5小时)

晚餐 (18:00-20:00)
- 推荐餐厅

晚间 (20:00-22:00)
- 夜景/夜市/演出
```

### 多日游模板

```
第1天: 抵达 + 周边探索
第2天: 核心景点 (必去)
第3天: 次要景点 + 体验
第4天: 周边小镇/自然
第5天: 购物 + 返程
```
